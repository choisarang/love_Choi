자바스크립트 기초

배열 이해하기

배열이란, 목록의 나열이라는 의미를 가지고 있는 형식.
이는 다양한 내용 및 변수 또는 함수를 나열하는 방식

예시 : 커피의 종류

  • 아메리카노
  • 카페라떼
  • 카페모카
  • 사케라토
  • 콜드브루

객체 이해하기

객체란, 배열과 유사하며 나열의 의미는 같으나 정의와 설명이 첨부된 형식.
정의의 설명값이 함수로 처리되면 메소드라고 불리며, 단순설명으로 처리되면 프로퍼티라고 불린다.

예시 : 음료의 종류

커피
커피원드를 주원료로 한 음료
쥬스
다양한 종류를 가진 커피를 제외한 음료
디저트
커피 및 쥬스와 곁들여 먹을 수 있는 다과
